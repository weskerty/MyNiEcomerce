## Requisitos


Queremos fomentar el consumo sostenible. Esta plataforma solo permite vendedores que cumplan criterios ambientales y de calidad.


### Contaminación y materiales

Se prohíben productos y empaques que generen impacto ambiental evitable, incluyendo materiales de alto residuo, difícil reciclaje o baja durabilidad.

#### Plásticos
No se permite el uso de plásticos de un solo uso ni desperdicio innecesario en empaques, presentación o distribución.

### Diseño no reparable
Ejemplo dispositivos electronicos de baja reparabilidad y bootloader cerrado que imposibilite cualquier metodo para darle nueva vida.
Ejemplo productos de Apple, Nuevos Samsung, Nuevos Xiaomis etc. Los Usados se permiten para no dejarlos como chatarra...

---

### Exclusividad de productos
Si ya existe un producto registrado (ej: “Torta de Chocolate Sin Lactosa”), no se permite otro igual. Pero si se permiten variantes como "Torta de Chocolate con Limon"
Se prioriza calidad, durabilidad y menor impacto ambiental.

### Alimentos

#### Vegetariano 
Si son alimentos deben cumplir la regla Vegetariana (no confundir con [vegano ↗️](https://es.wikipedia.org/wiki/Veganismo))

1. No carnes
2. Informacion Nutricional 
3. Informacion de Alergenicos 

No se permite carne, pero si huevo, leche, quesos, miel etc.


#### No platicos 
De igual manera que demas productos, envolturas plasticas no estan permitidas si usted no se encargara del reciclaje.
