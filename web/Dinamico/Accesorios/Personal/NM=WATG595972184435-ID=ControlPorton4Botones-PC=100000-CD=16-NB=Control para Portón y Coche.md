## Control Porton Universal 

<div id="cs"></div>

### Características
Copia controles
Frecuencia: 260MHz – 868MHz  
Compatible con portones eléctricos, alarmas, coches y motos (depende del sistema)  
Memoria interna para múltiples controles, 1 boton por cada control copiado.
Batería de larga duración  (Pila A23)

Pila facil de cambiar, si se agota solo se cambia la pila. Puedes traerla de vuelta para reemplazo de Pila

#### Compatibilidad
Clonación rápida presionando botones del control original  
Funciona con controles de código fijo (fixed code)  
NO compatible con sistemas de código cambiante (rolling code)  

Muchos coches modernos tienen codigo cambiante del control, esto para evitar ser copiados. Este control solo copia la señal y no el metodo de seguridad del control, es por eso que la compatibilidad dependera del tipo de sistema que se tiene.
