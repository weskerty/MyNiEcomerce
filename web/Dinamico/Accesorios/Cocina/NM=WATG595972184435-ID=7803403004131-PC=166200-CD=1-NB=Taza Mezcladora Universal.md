## Taza Auto Mezcladora USB con Base Magnética

<div id="cs"></div>

### Características
#### Taza
> Sistema de mezcla automática integrado  
Carga por USB (sin pilas desechables)  
Material apto para bebidas calientes y frías  
Fácil limpieza  
Diseño portátil  

#### Base Magnética
> Base desacoplable tipo “agitador magnético”  
Funciona con otras tazas, vasos etc.  
Pastilla imantada interna (stir bar)  
Mezcla uniforme sin cucharas  
Motor silencioso y eficiente  

---

## 🌍 ¿Por qué elegir este tipo de taza?

- Reutilizable, sin pilas → menos residuos  
- Permite usar cualquier taza compatible → mayor vida útil  
- Reduce uso de utensilios desechables  
- Bajo consumo energético (USB)

### Cuidados
- Evitar líquidos muy densos que bloqueen el giro, pueden quemar el motor.
- Si la bateria deja de funcionar no lo tires, se puede reparar. Traelo a la tienda para un cambio de bateria.
